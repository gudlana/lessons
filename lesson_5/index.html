Підключення шрифтів

Варіант 1 - Google Fonts

Розглянемо два варіанти підключення - за допомогою сервісу та плагіну

Google Fonts - це сервіс, за адресою fonts.google.com, або база даних шрифтів, які ми можемо приєднати до проекту.
Вибираємо шрифт з необхідними членами родини шрифту та обираємо тип підключення за допомогою тегу link в html або через @import в файл стилю css

  • Наприклад підключення шрифту родини "Inter"
    • за допомогою тегу link - < link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
    • через @import в файл стилю css - @import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap');

Плагін Google Fonts інсталюємо в VS Code

Використання плагіну Google Fonts дозволяє більш швидко підключати шрифти

Натискаємо F1 (визиваємо командний рядок для усіх встановлених плагінів) -> шукаємо по назві плагіну (Google Fonts) -> обираємо варіант підключення (в html або css) -> шукаємо по назві шрифта наприклад Inter -> обираємо членів родини які необхідні. Використання плагіна більш пріоритетне.

Доречі, до цього тексту застосований шрифт родини Inter, підключений за допомогою сервісу Google Fonts та вказаний як основний шрифт на сторінці.

Варіант 2 - підключення вручну

Такий спосіб застосовується у разі відсутності шрифта у базі Goolge Fonts. Набагато складніший варіант.

По-перше такий шрифт має бути наданий замовником або дизайнером. У випадку коли його немає, шукаємо в інтернеті. Завантажені шрифти у форматі ttf не можна використовувати, їх потрібно конвертувати у веб шрифти. Для цього є спеціальні генератори, наприклад https://www.whatfontis.com/Webfont-Generator.html. Отримаємо файли з розширенням woff та woff2 для кожного з шрифтів.
Далі в файлі стилів прописуємо наступну конструкцію для кожного з шрифтів:

  • @font-face {
    • font-family: "Causten";
    • src:
      • url("../fonts/Causten-Regular.woff2") format("woff2"),
      • url("../fonts/Causten-Bold.woff") format("woff");
    • font-weight: normal;
    • font-style: normal;
    • font-display: swap;
  • }

До цього розділу застосовується шрифт "Causten", підключений handmade варіантом

CSS властивості шрифтів

font-family

Встановлює родину шрифту
font-family: "Ім'я шрифту", "Ім'я шрифту", тип шрифту;
успадковується

  • Типи шрифтів:
    • serif - шрифт з зарубками (засічками)
    • sans-serif - рублений (без зарубок)
    • cursive - курсивні шрифти
    • fantasy - декоративні
    • monospace - моноширинні

font-size

Встановлює розмір шрифту
font-size: цифрове значення одиниця виміру;
(за замовчуванням 16px, якщо в браузері не вказано іншого)
успадковується
Цей текст відображається зі значенням властивості font-size 20px;

font-style

Встановлює стиль шрифту
font-style: назва стилю
успадковується

  • normal (за замовчуванням) - звичайний (не змінений, стандартний) стиль
  • italic - курсив
  • oblique - нахилення літер

font-weight

Встановлює насиченість (товщину) шрифту
font-weight: цифрове значення або значення словом
успадковується

  • 100 - thin
  • 300 - light
  • 400 - Regular (normal) (за замовчуванням)
  • 500 - medium
  • 600 - semibold
  • 700 - bold
  • 900 - black

CSS властивості тексту

color

Встановлює колір тексту
color: слово або код кольору
успадковується

text-align

Встановлює вирівнювання тексту (та рядкових, блочно-рядкових елементів)
text-align: значення
успадковується (текстом)

  • center - вирівнює по центру
  • left (за замовченням)- вирівнює по лівому краю
  • right - вирівнює по правому краю
  • justify - вирівнює текст по ширині об'єкту в якому знаходиться автоматично змінюючи пробіли між словами

text-decoration

Додає тексту певне оформлення
text-decoration: значення
успадковується (текстом)

  • line-through - закреслює текст
  • overline - лінія над текстом
  • underline - лінія під текстом
  • none (за замовченням) - відміняє змінений параметр

text-shadow

Додає тексту тінь
text-shadow: горизонтальне_зміщення вертикальне_зміщення розмір колір
успадковується (текстом) може бути мультитінь (значеняя через кому)

text-transform

Встановлює тип (регістр) літер (прописні або заглавні)
text-transform: значення
успадковується (текстом)

  • capitalize - кожне слово починається з заглавної літери
  • lowercase - усі літери прописні (нижній регістр)
  • uppercase - усі літери заглавні (верхній регістр)
  • none (за замовченням) - відміняє змінене значення

line-height

Визначає міжрядковий інтервал (відстань) (висота рядка)
line-height: значення;
Перевести пікселі в частини:
значення line-height поділити на значення font-size
успадковується (текстом, лише якщо задані частини)

letter-spacing

Встановлює інтервал між літерами (символами)
letter-spacing: значення;
успадковується (текстом)

word-spacing

Встановлює інтервал між словами
word-spacing: значення;
успадковується (текстом)

text-indent

Червоний рядок. Встановлює відступ у першого рядка тексту
text-indent: значення;
успадковується (текстом)

white-space

Керує поведінкою пробілів між словами
white-space: значення;
nowrap - забороняє переніс рядка
normal - (за замовчуванням) не змінює (або відміняє) значення
успадковується (текстом)
